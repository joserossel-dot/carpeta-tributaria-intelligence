# Carpeta Tributaria
## Objetivo
Desarrollar una plataforma capaz de extraer, estructurar y unificar la información contenida en la Carpeta Tributaria Electrónica emitida por el Servicio de Impuestos Internos (SII) de Chile.
El sistema deberá transformar documentos PDF en un modelo de datos estructurado que pueda ser utilizado por aplicaciones, APIs, procesos de análisis e inteligencia artificial.
## Objetivos específicos
* Extraer automáticamente la información de la Carpeta Tributaria.
* Normalizar los datos bajo un modelo único.
* Exportar la información a JSON, Excel y API.
* Permitir comparar distintas Carpetas Tributarias.
* Servir como base para futuros proyectos tributarios.
